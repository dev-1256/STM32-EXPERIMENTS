# Experiment 08 — PWM Servo Motor Control

## Aim
To generate PWM signals using a timer and control the angular position of a servo motor.

## Apparatus
- STM32 Discovery Kit
- STM32CubeIDE
- STM32CubeMX
- Servo Motor
- Connecting wires

## Theory
PWM signals are used to control the rotational angle of servo motors.

## Procedure
1. Configure timer in PWM mode.
2. Set PWM frequency and duty cycle.
3. Connect servo motor.
4. Change duty cycle to rotate servo.

## Result
Servo motor rotated according to PWM duty cycle.
