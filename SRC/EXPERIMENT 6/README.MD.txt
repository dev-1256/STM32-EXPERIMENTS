# Experiment 06 — LSM6DSL Accelerometer using SPI

## Aim
To configure SPI and read three-axis acceleration data from the LSM6DSL sensor.

## Apparatus
- STM32 Discovery Kit
- STM32CubeIDE
- STM32CubeMX
- RealTerm

## Theory
The LSM6DSL sensor provides accelerometer data through SPI communication.

## Procedure
1. Configure SPI peripheral.
2. Initialize LSM6DSL sensor.
3. Read X, Y, and Z acceleration values.
4. Display sensor readings on terminal.

## Result
Three-axis acceleration data displayed successfully.
