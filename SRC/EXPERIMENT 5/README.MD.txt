# Experiment 05 — LPS22HB Pressure Sensor

## Aim
To interface the LPS22HB sensor via I2C and obtain pressure data from environmental monitoring.

## Apparatus
- STM32 Discovery Kit
- STM32CubeIDE
- STM32CubeMX
- RealTerm

## Theory
LPS22HB is a digital pressure sensor communicating through I2C interface.

## Procedure
1. Configure I2C communication.
2. Initialize LPS22HB sensor.
3. Read pressure values.
4. Print values using USART terminal.

## Result
Pressure sensor data obtained successfully.
