# Experiment 04 — HTS221 Temperature and Humidity Sensor

## Aim
To interface the HTS221 sensor via I2C and acquire temperature and humidity measurements.

## Apparatus
- STM32 Discovery Kit
- STM32CubeIDE
- STM32CubeMX
- RealTerm

## Theory
HTS221 is an environmental sensor capable of measuring humidity and temperature using I2C communication.

## Procedure
1. Enable I2C peripheral.
2. Initialize HTS221 sensor.
3. Read temperature and humidity values.
4. Display values on serial terminal.

## Result
Temperature and humidity values acquired successfully.
