# Experiment 07 — SSD1306 OLED Display Interfacing

## Aim
To interface an SSD1306 display over I2C and present real-time sensor values on-screen.

## Apparatus
- STM32 Discovery Kit
- STM32CubeIDE
- STM32CubeMX
- SSD1306 OLED Display
- Connecting wires

## Theory
SSD1306 OLED displays use I2C communication for displaying text and graphics.

## Procedure
1. Configure I2C peripheral.
2. Initialize SSD1306 display.
3. Read sensor data.
4. Display values on OLED screen.

## Result
Sensor values displayed successfully on OLED display.
