\# STM32 IoT Lab Experiments Repository



\## Project Overview



This repository contains STM32-based IoT laboratory experiments implemented using STM32CubeIDE and STM32CubeMX.



The project demonstrates:

\- GPIO interfacing

\- USART communication

\- I2C sensor interfacing

\- SPI communication

\- OLED display interfacing

\- PWM motor control

\- Wi-Fi communication

\- BLE advertising



All experiments were implemented on the STM32 Discovery Kit.



\---



\# Hardware Used



\- STM32 Discovery Kit

\- HTS221 Temperature \& Humidity Sensor

\- LPS22HB Pressure Sensor

\- LSM6DSL Accelerometer

\- SSD1306 OLED Display

\- Servo Motor

\- Wi-Fi Module

\- BLE Module
